# Study

# Rule
월, 수, 금 : 해당 주차 알고리즘 문제 풀이
화, 목 : CS강의 시청

12 ~ 1 : 알고리즘 블로그 정리 개념 설명 + 일주일간 코드 내용 리뷰 
1 ~ 2 : 금주의 도전 알고리즘 문제
2 ~ 3 : 푼 알고리즘 문제 리뷰
3 ~ 4 : cs 문제풀이 + 질의응답


CS 문제는 금주에 학습한 내용으로 문제 3문제, 답 정리해서 토요일 밤까지
정유준 << 에게 보내주시면 됩니다.

다음주 미참여 인원 일요일까지
불참 패널티 : 과제 알고리즘 1문제 2000원, cs 정리 1개당 2000원
특별한 사유를 제외하고 3회 결석시 탈퇴
패널티 벌금은 다음주 스터디룸 예약에 사용
